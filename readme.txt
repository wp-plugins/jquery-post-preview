== Description ==

Live post preview on "Write/Edit post" page of WordPress admin area using jQuery. If you don't like WordPress built-in Visual Editor (because it puts a lot of unnecessary tags or by any other reasons), but you want to have live post preview like in Visual Editor, this plugin may be useful for you.

== Installation ==

1. Upload `jquery-post-preview` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Deactivate Visual Editor in your user profile.
4. That's all.

You'll find "Preview" button on "Write/Edit post" page above the post text field.
